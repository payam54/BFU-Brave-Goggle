# BFU-Brave-Goggle
A Brave Search Goggle that excludes social media, event, and ticketing websites (and all of their sub-domains) from search results.

## Excluded Websites

### Social Media
- **yelp.com**
- **facebook.com**
- **tiktok.com**
- **instagram.com**
- **x.com**
- **youtube.com**

### Event Platforms
- **eventbrite.com**
- **meetup.com**
- **nextdoor.com**

### Ticketing Platforms
- **ticketmaster.com** 

## Usage

To use this Goggle in Brave Search:
1. Open [Brave Search](https://search.brave.com).
2. Go to **Settings → Goggles**.
3. Click **Add a Goggle by URL** and paste the raw URL of `BFU.goggle` from this repository.
